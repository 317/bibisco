Read me
-------

bibisco doesn't need to be installed.
Launch bibisco.exe and enjoy.

Note: bibisco is a java application and ships its own JRE. You don't need to install or update JRE.

Note: if you have a previous version of bibisco installed on mac, don't unzip this version in the same folder of the previous version.

Note: don't unzip bibisco in a directory containing a non-ASCII characters (i.e. accented letters) in the name.
